Standard JS Functions

最后更新: 2026-03-07

Console

  • console_log(...args) - 输出普通日志
    示例: console_log('Hello', 'World', 123);
  • console_info(...args) - 输出提示信息
    示例: console_info('Process started');
  • console_warn(...args) - 输出警告信息
    示例: console_warn('Deprecated function');
  • console_error(...args) - 输出错误信息
    示例: console_error('Failed:', error.message);
  • console_debug(...args) - 输出调试信息
    示例: console_debug('Current value:', value);
  • console_clear() - 清空控制台
    示例: console_clear();

Base64

  • base64_encode(str) - 字符串转Base64
    示例: base64_encode('Hello World') = "SGVsbG8gV29ybGQ="
  • base64_decode(str) - Base64解码
    示例: base64_decode('SGVsbG8gV29ybGQ=') = "Hello World"

JSON

  • json_parse(str) - JSON转对象
    示例: json_parse('{"name":"John"}') = { name: "John" }
  • json_stringify(obj) - 对象转JSON
    示例: json_stringify({name: "John"}) = '{"name":"John"}'
  • json_format(str) - 格式化JSON
    示例: json_format('{"name":"John"}') = 格式化后的多行JSON

String

  • string_format(fmt, args) - 字符串格式化
    示例: string_format('Hello {0}', ['World']) = "Hello World"
  • string_join(sep, arr) - 连接数组
    示例: string_join(',', [1,2,3,4]) = "1,2,3,4"

Math

  • math_sum(arr) - 求和
    示例: math_sum([1,2,3,4,5]) = 15
  • math_avg(arr) - 平均值
    示例: math_avg([1,2,3,4,5]) = 3

DateTime

日期时间拼接

  • datetime_combine(datePart, timePart, format?) - 将日期和时间拼接，可指定返回格式
    示例: datetime_combine('2026-03-03', '13:30:00') = DateTime对象
    示例: datetime_combine('2026-03-03', '13:30', 'yyyy-MM-dd HH:mm') = "2026-01-01 13:30"
  • datetime_combine_now(datePart, format?) - 将指定日期与当前时间拼接
    示例: datetime_combine_now('2026-01-01') = 2026-01-01 + 当前时间
  • datetime_combine_today(timePart, format?) - 将指定时间与今天日期拼接
    示例: datetime_combine_today('14:30:00') = 今天日期 + 14:30:00
  • datetime_to_string(dateTimeObj, format) - 将DateTime对象转换为指定格式字符串
    示例: datetime_to_string(datetime_combine('2026-01-01', '13:30'), 'yyyy年MM月dd日 HH:mm') = "2026年01月01日 13:30"

当前时间

  • datetime_now() - 获取当前时间字符串
    示例: datetime_now() = "2026-01-01 12:34:56"
  • datetime_today() - 获取当前日期字符串
    示例: datetime_today() = "2026-01-01"
  • datetime_timestamp() - 获取当前Unix时间戳（秒）
    示例: datetime_timestamp() = 1704111896
  • datetime_timestamp_ms() - 获取当前Unix时间戳（毫秒）
    示例: datetime_timestamp_ms() = 1704111896123

格式化与解析

  • datetime_format(datetime, format) - 格式化日期时间
    示例: datetime_format('2026-01-01 13:30:45', 'yyyy/MM/dd HH:mm') = "2026/01/01 13:30"
  • datetime_parse(dateString, format) - 解析字符串为日期时间
    示例: datetime_parse('2026/01/01', 'yyyy/MM/dd') = "2026-01-01 00:00:00"

日期计算

  • datetime_add(dateString, days) - 日期加减天数
    示例: datetime_add('2026-01-01', 5) = "2026-01-06"
  • datetime_add_months(dateString, months) - 日期加减月份
    示例: datetime_add_months('2026-01-01', 2) = "2026-03-01"
  • datetime_diff(startDate, endDate, unit) - 计算时间差（unit: days/hours/minutes/seconds）
    示例: datetime_diff('2026-01-01', '2026-01-05', 'days') = 4

日期提取

  • datetime_get_year(dateString) - 获取年份
    示例: datetime_get_year('2026-01-01') = 2026
  • datetime_get_month(dateString) - 获取月份
    示例: datetime_get_month('2026-01-01') = 1
  • datetime_get_day(dateString) - 获取日期
    示例: datetime_get_day('2026-01-01') = 1
  • datetime_get_hour(dateString) - 获取小时
    示例: datetime_get_hour('2026-01-01 13:30') = 13
  • datetime_get_minute(dateString) - 获取分钟
    示例: datetime_get_minute('2026-01-01 13:30') = 30
  • datetime_get_second(dateString) - 获取秒
    示例: datetime_get_second('2026-01-01 13:30:45') = 45
